fundamentals of css

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It allows you to control the layout, colors, fonts, and overall appearance of web pages.

1. Inline CSS

Inline CSS is used to apply a unique style to a single HTML element using the style attribute directly within the element.

  • <p style="color: red;">Red text</p>
  • <h1 style="font-size: 32px;">Large heading</h1>
  • <div style="background: yellow;">Yellow background</div>
  • <span style="font-weight: bold;">Bold text</span>
  • <a style="text-decoration: none;">No underline link</a>

2. Internal CSS

Internal CSS is defined within the <style> tag in the <head> section of an HTML document and affects elements on that page.

  • body { background-color: #f0f0f0; }
  • h2 { color: blue; }
  • p { line-height: 1.5; }
  • ul { list-style-type: square; }
  • a { font-style: italic; }

3. External CSS

External CSS is written in a separate file (e.g., style.css) and linked to the HTML document, allowing styles to be reused across multiple pages.

  • Linking with <link rel="stylesheet" href="style.css">
  • body { margin: 0; }
  • nav { background: #333; }
  • footer { text-align: center; }
  • img { max-width: 100%; }

CSS Frameworks

CSS frameworks are pre-prepared libraries that make web development faster and easier by providing ready-to-use styles and components.

  • Bootstrap
  • Tailwind CSS
  • Bulma
  • Foundation
  • Materialize

Summary

CSS can be applied in various ways: inline, internal, external, and through frameworks. Each method has its own use cases and benefits, helping developers style web pages efficiently.
